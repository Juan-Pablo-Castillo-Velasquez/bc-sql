# Semana 02 — DDL: Diseño de Esquemas

## Objetivos

- Crear tablas con columnas, tipos de datos y constraints usando `CREATE TABLE`
- Aplicar restricciones `NOT NULL`, `DEFAULT`, `UNIQUE` y `CHECK`
- Modificar y eliminar tablas con `ALTER TABLE` y `DROP TABLE`
