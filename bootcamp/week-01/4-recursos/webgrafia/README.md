## Documentación Oficial

### SQLite — Documentación oficial

La referencia definitiva de SQLite. Esta semana consultar:
`About SQLite` y `SQL As Understood By SQLite → SELECT`.

- **URL**: https://www.sqlite.org/docs.html

## Tutoriales Interactivos

### W3Schools — SQL SELECT

Tutorial paso a paso de `SELECT`, `WHERE` y `ORDER BY` con editor en línea
para practicar sin instalar nada.

- **URL**: https://www.w3schools.com/sql/sql_select.asp

### SQLZoo — SELECT basics

Ejercicios interactivos en el navegador. Completar la sección
**"SELECT basics"** (lecciones 0, 1 y 2).

- **URL**: https://sqlzoo.net/wiki/SELECT_basics

## Sandbox Online

### DB Fiddle — SQLite sandbox

Entorno online para ejecutar SQL sin instalar nada. Seleccionar
**SQLite 3.39** como motor.

- **URL**: https://www.db-fiddle.com/
