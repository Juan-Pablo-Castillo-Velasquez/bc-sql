# 📚 Ebooks Gratuitos — Semana 01

Lecturas recomendadas para reforzar los fundamentos de bases de datos relacionales.

---

## 1. Learning SQL — Alan Beaulieu (O'Reilly)

**Relevancia**: Capítulos 1–2 cubren exactamente los temas de esta semana: qué es
una base de datos relacional, tablas, tipos de datos y tu primera consulta `SELECT`.

- **Formato**: Vista previa gratuita en O'Reilly
- **URL**: https://www.oreilly.com/library/view/learning-sql-3rd/9781492057604/

---

## 2. SQLite Tutorial — sqlitetutorial.net

**Relevancia**: Tutorial completo y gratuito de SQLite. La sección
"Getting Started" y "SQLite SELECT" son lectura obligatoria para esta semana.

- **Formato**: Web (HTML)
- **URL**: https://www.sqlitetutorial.net/

---

## 3. SQL Notes for Professionals — GoalKicker

**Relevancia**: Compendio gratuito de SQL. Revisar el **Capítulo 1**
(Getting started with SQL) para reforzar conceptos de tablas y SELECT básico.

- **Formato**: PDF gratuito
- **URL**: https://goalkicker.com/SQLBook/

---

> 💡 No es necesario leer todo el libro. Lee solo los capítulos indicados.
