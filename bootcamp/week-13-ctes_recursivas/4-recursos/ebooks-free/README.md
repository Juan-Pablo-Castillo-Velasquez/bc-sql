# Ebooks gratuitos — Semana 13: CTEs Recursivas

## Recomendados

### 1. Learning SQL (Alan Beaulieu) — O'Reilly

El capítulo 14 incluye una sección específica sobre CTEs recursivas
con ejemplos de jerarquías de empleados y generación de fechas.

🔗 https://openlibrary.org/books/OL24202440M

### 2. PostgreSQL: Up and Running (Regina Obe, Leo Hsu)

Cubre CTEs recursivos en el contexto de PostgreSQL, incluyendo
el control de profundidad con `WHERE depth <` y detección de ciclos.

🔗 https://openlibrary.org/works/OL17127861W

### 3. The Art of PostgreSQL (Dimitri Fontaine)

Capítulo dedicado a datos jerárquicos con CTEs recursivos, incluyendo
el modelado de árboles y el uso de `ltree` como extensión alternativa.

🔗 https://theartofpostgresql.com/
