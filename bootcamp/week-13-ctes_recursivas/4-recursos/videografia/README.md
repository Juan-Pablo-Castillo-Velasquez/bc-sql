# Videografía — Semana 13: CTEs Recursivas

## Videos recomendados

### 1. Recursive CTEs in SQL — Fireship (YouTube)

Explica las CTEs recursivas con animaciones claras mostrando el árbol
de ejecución. Incluye ejemplos de jerarquías y generación de series.
Duración: ~12 min.

🔗 https://www.youtube.com/watch?v=7hZYh9qB3k4

### 2. SQL Recursive CTE — Alex the Analyst (YouTube)

Tutorial paso a paso con un ejemplo de árbol organizacional completo:
setup de tabla auto-referencial, caso base y caso recursivo explicados.
Duración: ~20 min.

🔗 https://www.youtube.com/watch?v=8Noh4rYPbRM

### 3. generate_series PostgreSQL — Crunchy Data (YouTube)

Muestra cómo usar `generate_series()` para generar calendarios,
rangos numéricos y series de tiempo en PostgreSQL eficientemente.
Duración: ~15 min.

🔗 https://www.youtube.com/watch?v=aaCRCPsCUCY
