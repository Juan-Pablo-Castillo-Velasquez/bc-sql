# Semana 07 — NULL y Constraints

## Objetivos

1. Entender que NULL representa un valor desconocido, no cero ni vacío
2. Filtrar y transformar NULLs con `IS NULL`, `COALESCE` y `NULLIF`
3. Aplicar `NOT NULL`, `UNIQUE`, `CHECK`, `PRIMARY KEY` y `FOREIGN KEY`
