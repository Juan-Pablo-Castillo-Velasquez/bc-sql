# Webgrafía — Semana 09: JOINs

## Referencias web

### 1. PostgreSQL Docs — JOIN Types

Documentación oficial de PostgreSQL con definición formal de cada tipo de
JOIN, incluyendo INNER, LEFT, RIGHT, FULL OUTER y CROSS.

🔗 https://www.postgresql.org/docs/current/queries-table-expressions.html#QUERIES-JOIN

### 2. W3Schools — SQL Joins

Referencia rápida con ejemplos interactivos para cada tipo de JOIN. Útil
como consulta rápida durante la práctica.

🔗 https://www.w3schools.com/sql/sql_join.asp

### 3. SQL Join Visualizer — Joins.spathon.com

Herramienta interactiva que muestra visualmente qué filas devuelve cada tipo
de JOIN con tablas de ejemplo editables.

🔗 https://joins.spathon.com/
