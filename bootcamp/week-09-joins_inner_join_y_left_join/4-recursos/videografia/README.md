# Videografía — Semana 09: JOINs

## Videos recomendados

### 1. SQL Joins Explained — Programming with Mosh (YouTube)

Explicación visual de todos los tipos de JOIN con diagramas de Venn. Cubre
INNER, LEFT, RIGHT y FULL OUTER JOIN con ejemplos prácticos.
Duración: ~30 min.

🔗 https://www.youtube.com/watch?v=2HVMiPPuPIM

### 2. SQL JOINs Tutorial — freeCodeCamp (YouTube)

Curso completo sobre JOINs con ejemplos en tiempo real. Incluye casos de uso
reales y errores comunes al combinar tablas.
Duración: ~60 min.

🔗 https://www.youtube.com/watch?v=9yeOJ0ZMUYw

### 3. Visual Explanation of SQL Joins — ByteByteGo

Animación que muestra visualmente cómo cada JOIN filtra filas. Ideal para
entender la diferencia entre INNER y LEFT JOIN en segundos.

🔗 https://blog.bytebytego.com/p/visual-explanation-of-sql-joins
