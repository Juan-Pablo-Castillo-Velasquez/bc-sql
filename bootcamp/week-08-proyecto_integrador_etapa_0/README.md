# Semana 08 — Proyecto Integrador: Etapa 0

## Objetivos

1. Integrar todos los conceptos de las semanas 01–07 en un solo esquema
2. Diseñar un esquema relacional normalizado de 3 tablas para tu dominio
3. Escribir consultas que combinen DDL, DML, filtros, agregación y NULL

## Esta semana es diferente

La semana 08 es el **capstone de la Etapa 0** (Fundamentos de SQL). No hay
conceptos nuevos. El tiempo completo se dedica a consolidar lo aprendido
mediante un proyecto integrador más exigente.
