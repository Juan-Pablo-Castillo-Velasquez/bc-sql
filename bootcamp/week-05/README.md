# Semana 05 — Operadores y Filtros

## Objetivos

1. Usar operadores `BETWEEN`, `IN` y `NOT IN` para filtros compactos
2. Buscar patrones de texto con `LIKE` y comodines
3. Componer filtros complejos combinando múltiples operadores
