# Ebooks gratuitos — Semana 23

## Recursos de descarga libre

- **Use The Index, Luke** — Markus Winand  
  El recurso definitivo sobre índices en SQL, disponible gratuitamente.  
  https://use-the-index-luke.com/

- **PostgreSQL: Up and Running** — O'Reilly  
  Capítulo 10: Performance y EXPLAIN ANALYZE.  
  Buscar en [Archive.org](https://archive.org) o la biblioteca de tu institución.

- **High Performance PostgreSQL for Rails** — Andrew Atkinson  
  Explica EXPLAIN, índices parciales y cubrientes con casos reales.  
  Demo disponible en: https://pgrailsbook.com/
