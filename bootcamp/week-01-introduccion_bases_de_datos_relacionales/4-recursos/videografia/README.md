# 🎬 Videografía — Semana 01

Videos recomendados para complementar la teoría de esta semana.

---

## 1. SQL Tutorial for Beginners — freeCodeCamp

**Duración**: 4h (ver solo los primeros 30 min esta semana)  
**Relevancia**: Introduce qué es SQL, cómo funciona una BD relacional y las
primeras consultas `SELECT`. Es el recurso más completo para principiantes.

- **Canal**: freeCodeCamp.org (YouTube)
- **URL**: https://www.youtube.com/watch?v=HXV3zeQKqGY

---

## 2. Relational Databases Explained in 4 Minutes — Fireship

**Duración**: 4 min  
**Relevancia**: Explicación visual y concisa de qué es una base de datos
relacional, tablas y relaciones. Ideal como repaso rápido.

- **Canal**: Fireship (YouTube)
- **URL**: https://www.youtube.com/watch?v=OqjJjpjDRLc

---

## 3. What is a Database? — Socratica

**Duración**: 5 min  
**Relevancia**: Explica de forma muy clara la diferencia entre datos, tablas
y bases de datos. Recomendado si el concepto aún no queda claro después de la teoría.

- **Canal**: Socratica (YouTube)
- **URL**: https://www.youtube.com/watch?v=Ls_LzOZ7x0c
