# Semana 01 — Introducción a Bases de Datos Relacionales

## Objetivos

- Comprender qué es una base de datos relacional y cómo organiza la información
- Identificar tablas, filas, columnas y tipos de datos básicos
- Escribir tu primera consulta `SELECT`
- Instalar y configurar SQLite para los ejercicios del bootcamp
