# Semana 10 — CROSS JOIN y SELF JOIN

## Tema

Tipos especiales de JOIN: el producto cartesiano (`CROSS JOIN`) y la unión
de una tabla consigo misma (`SELF JOIN`) para modelar relaciones jerárquicas.

## Objetivos

- Entender qué produce un CROSS JOIN y cuándo usarlo
- Escribir un SELF JOIN con aliases distintos para la misma tabla
- Modelar relaciones jefe → empleado con una columna auto-referencial
- Obtener reportes de dos niveles jerárquicos en una sola consulta
