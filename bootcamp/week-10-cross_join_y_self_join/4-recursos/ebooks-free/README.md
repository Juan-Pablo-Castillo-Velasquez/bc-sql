# Ebooks gratuitos — Semana 10: CROSS JOIN y SELF JOIN

## Recomendados

### 1. Learning SQL (Alan Beaulieu) — O'Reilly

El capítulo 5 incluye SELF JOIN y relaciones jerárquicas con ejemplos
prácticos de organigramas. Disponible en Open Library.

🔗 https://openlibrary.org/books/OL24202440M

### 2. SQL Antipatterns (Bill Karwin)

El capítulo sobre "Naive Trees" (Árboles ingenuos) explica las limitaciones
del SELF JOIN con múltiples niveles y cuándo usar CTEs recursivas.

🔗 https://pragprog.com/titles/bksqla/sql-antipatterns/

### 3. SQLite Documentation — JOIN

Referencia oficial sobre CROSS JOIN y SELF JOIN en SQLite con notas de
comportamiento y optimización.

🔗 https://www.sqlite.org/lang_select.html#join_clause
