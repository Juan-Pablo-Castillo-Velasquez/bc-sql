# Ebooks gratuitos — Semana 09: JOINs

## Recomendados

### 1. Learning SQL (Alan Beaulieu) — O'Reilly

Capítulos 5 y 6 cubren en profundidad los tipos de JOIN y cómo combinar
múltiples tablas. Disponible en Open Library.

🔗 https://openlibrary.org/books/OL24202440M

### 2. SQL for Data Scientists (Renée M. P. Teate)

Enfoque práctico en JOINs para análisis de datos. Capítulo 4 cubre LEFT JOIN
y casos de uso reales con datos faltantes.

🔗 https://sqlfordatascientists.com/

### 3. SQLite Documentation — Query Language

Referencia oficial sobre la sintaxis y comportamiento de cada tipo de JOIN
en SQLite (con notas sobre versiones requeridas).

🔗 https://www.sqlite.org/lang_select.html#join_clause
