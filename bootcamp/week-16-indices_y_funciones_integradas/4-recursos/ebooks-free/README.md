# Ebooks Gratuitos — Semana 16

## Use The Index, Luke — Markus Winand

Libro gratuito completo sobre índices SQL con foco en cómo el motor
los usa (o no los usa). Capítulos 1–3 son esenciales para esta semana.
https://use-the-index-luke.com/

## The Art of PostgreSQL — Dimitri Fontaine

Incluye capítulos sobre EXPLAIN, optimización con índices y funciones
integradas de PostgreSQL.
https://theartofpostgresql.com/

## Learning SQL — Alan Beaulieu

Cubre funciones de cadena, fecha y numéricas con ejemplos del mundo real.
Disponible en Open Library.
https://openlibrary.org/books/OL7387522M/Learning_SQL
