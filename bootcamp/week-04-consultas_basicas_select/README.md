# Semana 04 — Consultas Básicas con SELECT

## Objetivos

1. Seleccionar columnas específicas usando `SELECT` explícito
2. Filtrar filas con la cláusula `WHERE`
3. Ordenar resultados con `ORDER BY` y paginar con `LIMIT` / `OFFSET`
