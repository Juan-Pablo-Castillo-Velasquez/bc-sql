# Ebooks Gratuitos — Semana 15

## The Art of PostgreSQL — Dimitri Fontaine

Cubre window functions de navegación (LAG, LEAD) y vistas con ejemplos
de análisis de series temporales.
https://theartofpostgresql.com/

## Learning SQL — Alan Beaulieu

Capítulo 16 cubre analytic functions incluyendo FIRST_VALUE y LAST_VALUE
con ejemplos prácticos de reporting.
https://openlibrary.org/books/OL7387522M/Learning_SQL

## SQL for Data Analysis — Cathy Tanimura (O'Reilly)

Capítulos de análisis de cohortes y series temporales usan LAG/LEAD
intensivamente. Vista previa gratuita disponible.
https://www.oreilly.com/library/view/sql-for-data/9781492088776/
