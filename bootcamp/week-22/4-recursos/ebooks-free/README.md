# Ebooks gratuitos — Semana 22

## Recursos de descarga libre

- **PostgreSQL: Up and Running** — O'Reilly  
  Capítulo 13 cubre tipos JSON, operadores y funciones.  
  Buscar en [Archive.org](https://archive.org) o la biblioteca de tu institución.

- **Mastering PostgreSQL** — Hans-Jürgen Schönig  
  Explica JSONB, GIN y casos de uso reales en profundidad.  
  https://masteringpostgresql.com/

- **Practical SQL** — Anthony DeBarros  
  Capítulo sobre JSON y datos semiestructurados con ejemplos prácticos.  
  https://practicalsql.com/
